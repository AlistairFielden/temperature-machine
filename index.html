The homebrew data logger

View on GitHub

The temperature-machine is a low cost, do-it-yourself data logger built around the Raspberry Pi and the DS18D20 temperature sensor. You can track ambient temperature over days, weeks and months and display some pretty graphs.

Reactive

The front-end uses React to respond to temperature changes dynamically. See the latest temperatures without hitting 'refresh'

RESTful

A client/server architecture with a RESTful API to the backend

RRD

A fixed size (round robin) database stores and averages the temperature data over one, seven and thirty days
